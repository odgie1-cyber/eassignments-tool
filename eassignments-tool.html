eAssignments Deep Link Creator

The eAssignments Deep Link Creator is a new tool for staff to generate a direct URL for any assignment submission point. By simply entering the Submission ID, you can create a unique, clickable link that can be shared with students via email, Blackboard, or any other communication channel. This deep link bypasses the need for students to navigate help pages, taking them directly to the correct submission portal. It's important to note that only students who are a part of the marking group for that specific assignment will be able to access the link, ensuring secure and targeted access.
